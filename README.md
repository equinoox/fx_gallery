# 🎨 FX Gallery

🌐 Live Demo: https://fx-gallery.vercel.app/

FX Gallery is a modern, full-featured blog platform built with **Next.js**, **Tailwind CSS**, and **GraphQL**, powered by a headless CMS. It delivers a fast, responsive, and content-driven experience with dynamic data fetching and clean UI design.

---

## 📖 Overview

FX Gallery is designed as a scalable blog application that separates content management from the frontend. It integrates seamlessly with **GraphCMS** to fetch and display posts, categories, and user-generated comments in real time.

The application focuses on performance, usability, and maintainable architecture.

---

## ✨ Features

- 🏠 **Homepage with Featured Content**  
  Highlighted posts and intuitive category navigation

- 📝 **Dynamic Blog Posts**  
  Rich content rendering with author info and structured layout

- 🗂️ **Category Filtering**  
  Easily browse posts by category

- 🔗 **Related & Recent Posts**  
  Smart content suggestions to improve engagement

- 💬 **Comment System**  
  Submit comments via a custom Next.js API route

- 📱 **Responsive Design**  
  Fully optimized for mobile, tablet, and desktop

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ Next.js
- ⚛️ React
- 🎨 Tailwind CSS
- 💅 SCSS (via `sass`)

### Data & APIs
- 🔗 GraphQL (`graphql-request`)
