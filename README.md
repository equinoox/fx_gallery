 # FX Gallery

 FX Gallery is a modern blog application built with Next.js, Tailwind CSS, and GraphQL. It connects to a headless GraphCMS backend to fetch posts, categories, and comments, and includes a simple comment submission API.

 ## Features

 - Blog homepage with featured posts and category navigation
 - Full post pages with rich content, author details, and comments
 - Category filtering for post archives
 - Recent posts widget and related content suggestions
 - Comment submission through a Next.js API route
 - Responsive UI styled with Tailwind CSS and SCSS support

 ## Technology stack

 - Next.js
 - React
 - Tailwind CSS
 - GraphQL + `graphql-request`
 - GraphCMS as content source
 - `react-icons`, `react-multi-carousel`, `html-react-parser`
 - SCSS support via `sass`
